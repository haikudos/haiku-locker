Two keys, kept apart: the signer key lives on the node (step 1, more exposed); the manager-admin wallet is the funded, hardware-backed key you connect in steps 0/3/4. Never paste the signer private key into these pages — they only need the public key.

Opening these: double-clicking works for navigation, but wallet connection and API reads are most reliable when the folder is served over http:// (e.g. python3 -m http.server in this folder), since some wallets restrict file:// origins.
